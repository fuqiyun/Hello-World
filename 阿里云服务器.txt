linux 安装jdk

解压jdk  需要到系统盘 解压jdk  使用绝对路径  /usr/local  这种
tar -zxvf jdk.tar.gz

然后  /etc/profile  文件中加上  

#jdk
#jdk安装目录
export JAVA_HOME=/usr/local/jdk/jdk1.8.0_144 
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib:$CLASSPATH
export JAVA_PATH=${JAVA_HOME}/bin:${JRE_HOME}/bin
export PATH=$PATH:${JAVA_PATH}

然后 重新加载配置
source /etc/profile
最后 java -version

连接本地mysql提示无权限（ message from server: "Host 'fqy' is not allowed to connect to this MySQL server"）

解决方法:
打开 MySQL 配置文件（路径因操作系统而异）：

Linux: /etc/mysql/my.cnf 或 /etc/my.cnf
Windows: C:\ProgramData\MySQL\MySQL Server X.X\my.ini
找到 bind-address 配置项，将其改为：

bind-address = 0.0.0.0
 重启 MySQL 服务以应用更改
