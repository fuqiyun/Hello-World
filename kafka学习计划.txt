Kafka 学习任务计划
学习目标
理解 Kafka 的核心概念与架构
掌握 Kafka 的安装、配置与运维
熟练使用 Kafka 的 Java API 进行开发
了解 Kafka 的性能优化与监控
能够设计和实现基于 Kafka 的分布式系统

第一阶段：基础理论（1-2周）
学习内容
Kafka 简介与核心概念
什么是 Kafka
消息队列 vs 流处理平台
Kafka 的核心概念：Topic、Partition、Producer、Consumer、Broker
Kafka 架构原理
分布式架构设计
数据存储机制
副本与容错机制
消息传递语义

博客推荐：
https://www.confluent.io/blog/

https://www.iteye.com/blog/tag/Kafka
